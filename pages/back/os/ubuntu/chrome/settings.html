Почему chrome не открывает ссылку кликнутую в сторонних приложениях в Ubuntu?

Изменить параметр запуска

 
vim $HOME/.local/share/applications/google-chrome.desktop
# Exec=/opt/google/chrome/chrome -> Exec=/opt/google/chrome/chrome %U
